# Mobile-Robotics-DDPG-HER

> **Status: Work in Progress (WIP)**
> Овој проект е моментално во развој. Кодот се усовршува и дополнува.

Овој проект е изработен како дел од предметот **Мобилна роботика**. Целта е имплементација на алгоритам за учење со засилување (Reinforcement Learning) за управување со роботски манипулатор во средината `FetchReach-v4`.



## Детали за проектот
Фокусот е ставен на решавање на проблемот со „ретки награди“ (sparse rewards) во роботиката преку употреба на **Hindsight Experience Replay (HER)**, во комбинација со **Deep Deterministic Policy Gradient (DDPG)**.

Целиот систем (невронски мрежи, backpropagation, оптимизација) е напишан "од нула" користејќи **Python** и **NumPy**, без надворешни ML системи.

## Технички спецификации
- **Алгоритам:** DDPG + HER
- **Средина:** Gymnasium-Robotics (FetchReach)
- **Имплементација:** Pure NumPy
- **Статус:** Во развој (Ова е само првичен алгоритам и идејно решение)
